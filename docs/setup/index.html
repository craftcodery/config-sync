NorthBuilt
Workstation configuration in one command
Prerequisites
1Password desktop app installed with CLI integration enabled
(Settings → Developer → Integrate with 1Password CLI)
Run this in your terminal:
curl -fsSL https://setup.northbuilt.com/install | bash
What this installs
  • Homebrew (if not installed)
  • AWS CLI, jq, gum, glow
  • 1Password CLI integration
  • AWS credential helper (uses 1Password)
  • Hourly config sync service
After setup
Test your access with: aws sts get-caller-identity
