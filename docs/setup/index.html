Prerequisites

Run this in your terminal:

What this installs
  • Homebrew (if not installed)
  • AWS CLI, jq, gum, glow
  • 1Password CLI integration
  • AWS credential helper (uses 1Password)
  • Hourly config sync service
After setup
Test your access with: aws sts get-caller-identity
